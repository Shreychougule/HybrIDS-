# Datasets

This folder is reserved for local dataset files used by the project.

Large raw data files are intentionally not tracked in Git because GitHub has file size limits for normal repositories.

Current local files (not committed):
- `NF-UQ-NIDS-v2.csv`
- `archive.zip`
- `NF-UQ-NIDS-v2/NF-UQ-NIDS-v2.csv`

If you need to share large datasets, use one of these options:
- GitHub Releases (for smaller assets).
- Cloud storage (Google Drive, OneDrive, S3, etc.).
- Data hosting platforms (Kaggle, Zenodo, Hugging Face Datasets).
