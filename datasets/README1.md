# Datasets

This folder is reserved for local dataset files used by the project.

## NF-UQ-NIDS-v2 Network Intrusion Detection Dataset

`NF-UQ-NIDS-v2` is a network intrusion detection dataset used to train and evaluate this Hybrid NIDS project. It contains labeled network traffic records for normal activity and multiple attack categories, making it suitable for multi-class intrusion detection experiments.

Reference links:
- NF-UQ-NIDS-v2 Datasets: https://www.kaggle.com/datasets/aryashah2k/nfuqnidsv2-network-intrusion-detection-dataset
- archive.zip: https://drive.google.com/file/d/12FbASiYKxoxR1EHySoFCuZTM292gjY2g/view?usp=sharing
